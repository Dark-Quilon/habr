## Шаги деплоя

1. **Регистрация** → console.cloud.yandex.ru
2. **Создание ВМ** → 2 vCPU, 2 GB RAM, Ubuntu 22.04
3. **Подключение** → SSH через консоль
4. **Установка Docker** → curl get.docker.com
5. **Загрузка проекта** → git clone
6. **Настройка .env** → ALLOWED_HOSTS=<IP>
7. **Запуск** → docker compose up --build -d
8. **Открытие портов** → Security groups
