## Navbar (`Navbar.tsx`)

### Структура слева направо

```
[☰] Habr | Все потоки          [🔍] [✏️] [👤/Войти]
```

### Детали

**Цвет фона**: `#1f2937` (тёмно-синий)

**Бургер-кнопка** (`data-testid="burger-btn"`):
- При клике открывает offcanvas-меню
- Overlay с затемнением фона
- Клик на overlay закрывает меню

**Offcanvas-меню** (860px, фиксированное слева):
- Левая колонка (`#f7f8fa`): категории с иконками — Что нового, Все потоки, Разработка и инженерия (Бэкенд, Фронтенд, Мобильная разработка, Геймдев, Тестирование, AI и ML), Инфраструктура и данные
- Правая колонка: Технологии (Python, C++, Java, Go, PostgreSQL, Rust, Linux, Kotlin) и Темы (теги из API, сетка 4 колонки)

**Логотип**: "Habr" → `/`, рядом "| Все потоки"

**Иконка лупы** (`aria-label="Поиск"`, `data-testid="search-icon"`):
- При клике показывает inline строку поиска
- Сабмит → `router.push('/?search=...')`

**Иконка карандаша** (`aria-label="Написать публикацию"`, `data-testid="write-icon"`):
- Ведёт на `/write` (страница "Как стать автором")
- `title="Написать публикацию"` — нативный tooltip

**Иконка аккаунта** (авторизованный):
- Dropdown через React state (без Bootstrap JS)
- Пункты: Профиль, Уведомления, Выйти
- Badge с количеством непрочитанных уведомлений

**Кнопка "Войти"** (неавторизованный):
- Стиль `.habr-btn-login` — outline с border

## Главная страница (`page.tsx`)

Добавлена секция "Моя лента" над списком статей:

```tsx
// Авторизован
<div className="mb-4 p-3 border rounded">
  <h2>Моя лента</h2>
  <Link href="/feed">Перейти в ленту</Link>
</div>

// Не авторизован
<div className="mb-4 p-3 border rounded">
  <h2>Моя лента</h2>
  <p>Войдите, чтобы видеть свою ленту. <Link href="/login">Войти</Link></p>
</div>
```

Использует `getServerToken()` для определения авторизации на сервере.
