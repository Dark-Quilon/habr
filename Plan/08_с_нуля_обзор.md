# С нуля до текущего уровня проекта

## Что есть сейчас в проекте

- Django блог с авторизацией
- Статьи (черновик/опубликована), теги, Markdown
- Комментарии, голоса (лайк/дизлайк)
- Профили пользователей с аватарами
- Подписки на авторов, лента
- Уведомления
- Поиск, сортировка, пагинация
- Redis кэш
- Docker

## Главный принцип

Каждый этап — рабочий результат.
После каждого шага сайт запускается и что-то делает.
Не пытаться сделать всё сразу.
