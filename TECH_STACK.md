│                                             │

### Поток запросов:
1. **Браузер** → `localhost:3000` → Preact SPA
2. **Preact** → `/api/v1/*` → Django REST API (`localhost:8000`)
3. **Django** → Redis (кэш) → SQLite (данные)
4. **Ответ** → JSON → Preact → Рендер на клиенте
